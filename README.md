# TA Training Acknowledgement Site

This repository contains a static website created for **Assignment 1: Go Phish** in a Georgia Tech cybersecurity course.

## Purpose
The site simulates a **mandatory TA training acknowledgement page** as part of an academic phishing campaign.  
Its purpose is to demonstrate **social engineering and phishing campaign design**, not to perform a real attack.

## Scope & Limitations
- This site is **static** (HTML/CSS only)
- **No authentication** is required
- **No data is collected or stored**
- **No forms, tracking, or analytics** are implemented
- No malicious code is executed

Any references to logging or attacker infrastructure are **conceptual only** and described in the accompanying presentation slides.

## Structure
- `index.html` – Landing page for training acknowledgement
- `css/styles.css` – Site styling
- `assets/` - Stores assets like images
